🖥️ PowerShell
Bienvenido a mi sección dedicada a PowerShell, una de las herramientas más potentes para la automatización, administración de sistemas y gestión avanzada en entornos Windows, Linux y cloud.
Aquí recopilo mis apuntes, prácticas, comandos clave, scripts y metodologías para dominar PowerShell desde cero hasta un nivel profesional.

Este espacio está pensado tanto para mi propio aprendizaje como para cualquiera que quiera iniciarse o profundizar en la automatización con PowerShell.

📘 ¿Qué encontrarás en esta carpeta?
Los contenidos están organizados por temas, cada uno con su propio archivo en formato:

.md → lectura rápida desde GitHub

.docx → versión más visual para estudiar o imprimir
