**1\. QUÉ ES JAVA Y PARA QUE SE USA**

**☕ Qué es Java**

Java es un **lenguaje de programación orientado a objetos**, creado por Sun Microsystems en 1995 (hoy propiedad de Oracle). Su filosofía se resume en una idea clave: **“Write Once, Run Anywhere”** (escribe una vez, ejecútalo en cualquier lugar). Esto significa que un programa escrito en Java puede ejecutarse en distintos sistemas operativos sin necesidad de modificar el código.

**Características esenciales**

-   **Orientado a objetos** — organiza el código en clases y objetos, facilitando la reutilización y el mantenimiento.
-   **Multiplataforma** — gracias a la *Java Virtual Machine* (JVM), el mismo programa funciona en Windows, macOS, Linux, Android, etc.
-   **Seguro** — incorpora mecanismos para evitar accesos indebidos a memoria y ejecutar código en entornos controlados.
-   **Robusto** — maneja errores y excepciones de forma estructurada, reduciendo fallos en tiempo de ejecución.
-   **Multihilo** — permite ejecutar varias tareas a la vez dentro de un mismo programa.
-   **Gran ecosistema** — bibliotecas, frameworks y herramientas muy maduras.

**🧩 Cómo funciona Java internamente**

1.  El programador escribe código en archivos .java.
2.  El compilador lo transforma en **bytecode**, un formato intermedio.
3.  La **JVM** interpreta o compila ese bytecode para el sistema operativo donde se ejecuta.

Este proceso es lo que permite la portabilidad entre plataformas.

**🚀 Para qué se usa Java**

Java es uno de los lenguajes más versátiles del mundo. Se utiliza en:

**1\. Desarrollo de aplicaciones empresariales**

Es muy común en grandes empresas por su estabilidad y escalabilidad. Frameworks típicos: **Spring**, **Jakarta EE**.

**2\. Aplicaciones Android**

El desarrollo nativo de Android se basó durante años en Java, y aún hoy es uno de los lenguajes principales junto con Kotlin.

**3\. Aplicaciones web**

Java se usa para crear **backends** robustos que gestionan bases de datos, APIs y lógica de negocio.

**4\. Software de escritorio**

Aunque hoy es menos común, Java permite crear interfaces gráficas con **JavaFX** o **Swing**.

**5\. Sistemas financieros y banca**

Muchos bancos y sistemas de trading utilizan Java por su seguridad y rendimiento.

**6\. Big Data**

Herramientas como **Hadoop** están escritas en Java, lo que lo hace relevante en análisis de datos a gran escala.

**7\. Internet of Things (IoT)**

Java ME (Micro Edition) se usa en dispositivos pequeños y embebidos.

**🛠️ Ventajas y desventajas de Java**

**Ventajas**

-   Portabilidad real entre sistemas.
-   Comunidad enorme y mucha documentación.
-   Rendimiento sólido gracias a la JVM.
-   Ideal para proyectos grandes y de larga duración.
-   Ecosistema profesional muy consolidado.

**Desventajas**

-   Sintaxis más extensa que lenguajes modernos como Python o Kotlin.
-   Consumo de memoria mayor en comparación con lenguajes más ligeros.
-   No es la opción más rápida para prototipos rápidos.

**📚 Resumen para estudiar**

-   Java = lenguaje + plataforma.
-   Se ejecuta en la JVM → portabilidad.
-   Orientado a objetos, seguro y robusto.
-   Usado en empresas, Android, web, banca, Big Data y más.
-   Muy estable, pero algo verboso.
