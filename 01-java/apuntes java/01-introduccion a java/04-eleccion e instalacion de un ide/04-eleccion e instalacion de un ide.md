**4\. ELECCIÓN E INSTALACIÓN DE UN IDE**

**🧩 Qué es un IDE y por qué importa al programar en Java**

Un IDE (*Integrated Development Environment*) reúne en una sola herramienta el editor de código, el compilador, el depurador, el gestor de proyectos y extensiones. Elegir bien tu IDE afecta a:

-   La velocidad con la que escribes y refactorizas código.
-   La facilidad para depurar errores.
-   La integración con Maven, Gradle, Git y frameworks como Spring.
-   El rendimiento general del entorno de trabajo.

**🧠 Comparación general de los tres IDE principales**

**IntelliJ IDEA — Enfoque profesional y asistencia inteligente**

-   Reconocido como el IDE más usado por desarrolladores Java.
-   Ofrece análisis de código en tiempo real, refactorización avanzada y autocompletado muy preciso.
-   Disponible en versión **Community** (gratuita) y **Ultimate** (de pago).
-   Ideal para proyectos grandes, Spring Boot y desarrollo empresarial.

**Eclipse — Extensible y veterano**

-   Uno de los IDE más antiguos y estables del ecosistema Java.
-   Destaca por su enorme catálogo de plugins y su flexibilidad.
-   Adecuado para entornos académicos y proyectos que requieren personalización profunda.
-   Ligero en consumo de recursos comparado con IntelliJ.

**Visual Studio Code — Ligero y modular**

-   Editor rápido y minimalista que se convierte en IDE mediante extensiones.
-   Muy útil si trabajas con múltiples lenguajes además de Java.
-   Requiere instalar extensiones como *Extension Pack for Java*.
-   Popular por su rapidez y su marketplace de plugins.

**🛠️ Instalación de cada IDE paso a paso**

**Instalación de IntelliJ IDEA**

1.  Descargar desde la web de JetBrains (versión Community o Ultimate).
2.  Ejecutar el instalador y seguir el asistente.
3.  Elegir plugins iniciales (Java, Maven, Git).
4.  Abrir un proyecto o crear uno nuevo con Maven/Gradle.
5.  Configurar el JDK desde *Project Structure → SDKs*.

**Instalación de Eclipse**

1.  Descargar *Eclipse IDE for Java Developers*.
2.  Descomprimir o instalar según el sistema operativo.
3.  Seleccionar un *workspace* (carpeta de trabajo).
4.  Instalar plugins adicionales desde *Eclipse Marketplace*.
5.  Configurar el JDK en *Preferences → Java → Installed JREs*.

**Instalación de Visual Studio Code**

1.  Descargar VS Code desde la web oficial.
2.  Instalar el *Extension Pack for Java* (incluye soporte para Maven, Debugger y JDK Tools).
3.  Configurar el JDK en *Settings → Java: Home*.
4.  Instalar extensiones adicionales según tus necesidades (GitLens, Docker, Spring Tools).
5.  Abrir o crear un proyecto Java con Maven o Gradle.

**📌 Tabla comparativa para elegir IDE**

| Criterio | IntelliJ IDEA | Eclipse | VS Code |
| --- | --- | --- | --- |
| Rendimiento | Alto, pero consume más recursos | Medio | Muy alto, muy ligero |
| Facilidad de uso | Excelente | Media | Alta |
| Refactorización | Avanzada | Buena | Básica (mejora con extensiones) |
| Plugins | Muchos, de alta calidad | Muchísimos, muy variados | Enorme marketplace |
| Ideal para | Proyectos grandes, Spring | Personalización, entornos académicos | Multilenguaje, rapidez |

**🧭 Recomendación según tu perfil**

-   Si buscas **máxima productividad** y trabajarás con Spring o proyectos grandes → **IntelliJ IDEA**.
-   Si necesitas **flexibilidad extrema** y un entorno muy configurable → **Eclipse**.
-   Si prefieres algo **ligero, rápido y multiplataforma** → **VS Code**.

**📝 Resumen para estudiar**

-   Un IDE integra herramientas clave para programar en Java.
-   IntelliJ destaca por su inteligencia, Eclipse por su extensibilidad y VS Code por su ligereza.
-   La instalación en los tres casos es sencilla: descargar, instalar, configurar JDK y añadir plugins.
-   La elección depende del tipo de proyecto, tu experiencia y tus preferencias de trabajo.
