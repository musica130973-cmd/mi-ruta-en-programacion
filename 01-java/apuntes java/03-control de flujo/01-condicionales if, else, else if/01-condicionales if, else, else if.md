**🧠 PUNTOS IMPORTANTES**

-   Las condiciones se evalúan **de arriba hacia abajo**.
-   En cuanto una condición se cumple, **las demás se ignoran**.
-   else es opcional.
-   else if también es opcional, puedes tener solo if y else.
-   Las condiciones deben ser **booleanas** (true/false).

**🛠️ OPERADORES COMUNES EN CONDICIONES**

| **Operador** | **Significado** |
| --- | --- |
| == | Igual a |
| != | Distinto de |
| &gt; | Mayor que |
| &lt; | Menor que |
| &gt;= | Mayor o igual |
| &lt;= | Menor o igual |
| &amp;&amp; | Y lógico |
| \|\| | O lógico |

**🎓 RESUMEN FINAL**

-   **if** → evalúa una condición
-   **else if** → evalúa más condiciones si las anteriores fallan
-   **else** → se ejecuta cuando ninguna condición se cumple
-   Se usan para que el programa **decida qué hacer** según los datos
