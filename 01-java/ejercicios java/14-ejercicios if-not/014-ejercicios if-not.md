**14\. EJERCICIOS IF-NOT(!)**

**Ejercicio 1. Número no positivo**

Escribe un programa que pida un número entero por teclado y muestre un mensaje solo si el número NO es positivo.

![](./014-ejercicios%20if-not_images/image-001.png)

**Ejercicio 2. No es mayor de edad**

Pide la edad de una persona y muestra “No es mayor de edad” si NO tiene 18 años o más.

![](./014-ejercicios%20if-not_images/image-002.png)

  

**Ejercicio 3. No es par**

Escribe un programa que lea un número entero y muestre “El número NO es par” si el número no es par.

![](./014-ejercicios%20if-not_images/image-003.png)

**Ejercicio 4. Usuario no autorizado**

Define una variable boolean autorizado. Si no está autorizado (autorizado == false), el programa debe mostrar “Acceso denegado”.

**![](./014-ejercicios%20if-not_images/image-004.png)**

**Ejercicio 5. No está dentro del rango**

Pide un número y muestra “El número NO está entre 10 y 20” si el número no pertenece al intervalo cerrado \[10, 20\]

**![](./014-ejercicios%20if-not_images/image-005.png)**

**Ejercicio 6. No es vocal**

Pide un carácter por teclado y muestra “No es vocal” si el carácter no es una vocal minúscula.

![](./014-ejercicios%20if-not_images/image-006.png)

**Ejercicio 7. No está lloviendo**

Crea una variable booleana (lloviendo). Si no está lloviendo, muestra “Puedes salir sin paraguas”

![](./014-ejercicios%20if-not_images/image-007.png)

**Ejercicio 8. No coincide la contraseña**

Pide una contraseña y comprueba si no coincide con 1234. Si no coincide, muestra “Contraseña incorrecta”.

**![](./014-ejercicios%20if-not_images/image-008.png)**

**Ejercicio 9. No es fin de semana**

Pide un número del 1 al 7 representando el día de la semana.

Si el día no es sábado (6) ni domingo (7), muestra “Hoy toca trabajar”

![](./014-ejercicios%20if-not_images/image-009.png)

**Ejercicio 10. No hay stock**

Crea una variable int stock. Si no hay stock ( es decir, si stock == 0) , muestra “Producto agotado”.

![](./014-ejercicios%20if-not_images/image-010.png)
