**6\. EJERCICIOS INGRESAR DATOS JOPTIONPANE**

**Ejercicio 1. Pedir un nombre y saludar**

Pide al usuario su nombre y muestra un saludo personalizado

![](./06-ejercicios%20ingresar%20datos%20joptionpane_images/image-001.png)

**Ejercicio 2. Sumar dos números**

Pide dos números enteros y muestra la suma

![](./06-ejercicios%20ingresar%20datos%20joptionpane_images/image-002.png)

**Ejercicio 3. Área de un círculo**

Pide el radio y calcula el área de un círculo.

![](./06-ejercicios%20ingresar%20datos%20joptionpane_images/image-003.png)

**Ejercicio 4. Clasificar edad**

Pide la edad y clasifica: niño, adolescente, adulto o mayor

![](./06-ejercicios%20ingresar%20datos%20joptionpane_images/image-004.png)

**Ejercicio 5. Menú simple con opciones**

Muestra un menú con tres opciones: Sumar, Restar, Salir.

**![](./06-ejercicios%20ingresar%20datos%20joptionpane_images/image-005.png)**

**Ejercicio 6. Validar entrada numérica**

Pide un número hasta que el usuario introduzca un valor válido

![](./06-ejercicios%20ingresar%20datos%20joptionpane_images/image-006.png)

**Ejercicio 7. Conversor Celsius-Fahrenheit**

Convierte una temperatura de Celsius a Fahrenheit.

![](./06-ejercicios%20ingresar%20datos%20joptionpane_images/image-007.png)

**Ejercicio 8. Promedio de notas**

Pide tres notas y calcula el promedio

![](./06-ejercicios%20ingresar%20datos%20joptionpane_images/image-008.png)

**Ejercicio 9. Formulario básico**

Pide nombre, edad, ciudad y profesión.

![](./06-ejercicios%20ingresar%20datos%20joptionpane_images/image-009.png)

**Ejercicio 10. Login sencillo**

Pide usuario y contraseña y valida

![](./06-ejercicios%20ingresar%20datos%20joptionpane_images/image-010.png)

**Ejercicio 11. Selector de colores**

Muestra un menú desplegable con colores.

![](./06-ejercicios%20ingresar%20datos%20joptionpane_images/image-011.png)

**Ejercicio 12. Precio final con IVA**

Pide precio e IVA y calcula el total.

![](./06-ejercicios%20ingresar%20datos%20joptionpane_images/image-012.png)

**Ejercicio 13. Contador de caracteres**

Pide una frase y muestra cuántos caracteres tiene.

![](./06-ejercicios%20ingresar%20datos%20joptionpane_images/image-013.png)

**Ejercicio 14. Adivina el número**

El programa genera un número aleatorio del 1 al 10 y el usuario debe adivinarlo.

![](./06-ejercicios%20ingresar%20datos%20joptionpane_images/image-014.png)

**Ejercicio 15. Registro con confirmación**

Pide datos y pregunta si desea confirmar antes de mostrar el resumen.

**![](./06-ejercicios%20ingresar%20datos%20joptionpane_images/image-015.png)**
