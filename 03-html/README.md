☕ Apuntes de HTML
Bienvenido a la sección de HTML dentro de mi ruta de aprendizaje en programación.
Aquí recopilo todos mis apuntes, ejemplos y ejercicios relacionados con el marcado de páginas web, organizados de forma clara y progresiva para facilitar el estudio.

Este espacio está pensado tanto para mi propio aprendizaje como para cualquiera que quiera repasar los fundamentos de HTML de manera ordenada y práctica.

📘 ¿Qué encontrarás en esta carpeta?
Los apuntes están divididos por temas, cada uno con su propio archivo en formato:

.md → lectura rápida desde GitHub

.docx → versión más visual para estudiar o imprimir

Los contenidos incluyen, entre otros:

Introducción a HTML y su papel en la web

Relación entre HTML, CSS y JavaScript

Estructura básica de un documento HTML

Etiquetas de texto y semántica básica

Elementos de bloque y de línea

Enlaces y navegación

Imágenes y contenido multimedia

Listas y tablas

Formularios y validación nativa

Atributos globales y buenas prácticas

Accesibilidad (A11y)

Metadatos y SEO básico

HTML5 avanzado y APIs relacionadas

Prácticas y pequeños proyectos

Errores comunes y cómo evitarlos

🎯 Objetivo de estos apuntes
Mi intención es construir una base sólida en HTML, entendiendo no solo “qué etiqueta usar”, sino por qué y cuándo usarla, con especial foco en:

semántica

accesibilidad

buenas prácticas modernas

Este material me sirve como guía de estudio y como referencia rápida mientras maqueto páginas y practico con proyectos reales.
