-   <meta> y sus atributos
-   Etiquetas para SEO: title, meta description
-   Open Graph y tarjetas para redes sociales
-   Sitemap y robots (conceptos generales)

**📦 11. HTML5 avanzado**

-   Nuevos elementos semánticos
-   Formularios avanzados (nuevos tipos de input)
-   APIs del navegador relacionadas con HTML:
    -   Drag & Drop
    -   Web Storage (localStorage, sessionStorage)
    -   Geolocalización
    -   Multimedia y controles personalizados

**🧪 12. Prácticas y proyectos**

-   Crear una página personal
-   Crear un formulario completo con validación
-   Maquetar un blog con estructura semántica
-   Página responsive combinando HTML + CSS
-   Optimización de imágenes y carga

**🧭 13. Errores comunes y cómo evitarlos**

-   Mal uso de etiquetas semánticas
-   Falta de atributos alt
-   Anidar mal elementos
-   Usar HTML para estilos en vez de CSS
-   No validar el documento (W3C Validator)
