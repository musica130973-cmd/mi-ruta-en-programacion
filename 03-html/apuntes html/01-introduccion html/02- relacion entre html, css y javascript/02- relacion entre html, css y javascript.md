**2\. RELACION ENTRE HTML, CSS Y JAVASCRIPT**

En el desarrollo web, HTML, CSS y JavaScript trabajan juntos para construir páginas y aplicaciones. Cada uno cumple un rol distinto, pero se complementan para crear experiencias completas

**🧱 1. HTML: la estructura**

HTML (**HyperText Markup Language**) es el lenguaje que define **la estructura y el contenido** de una página web.

**Qué aporta:**

-   Organiza la información en elementos: títulos, párrafos, imágenes, botones, formularios.
-   Es el esqueleto de la web.
-   Sin HTML, no habría nada que mostrar.

**Ejemplo:**

![](./02-%20relacion%20entre%20html%2C%20css%20y%20javascript_images/image-001.png)

**🎨 2. CSS: el estilo**

CSS (**Cascading Style Sheets**) se encarga de **la apariencia visual** de la página.

**Qué aporta:**

-   Colores, tamaños, tipografías.
-   Distribución y diseño (layouts).
-   Animaciones y transiciones.
-   Adaptación a móviles (responsive design).

**Ejemplo:**

![](./02-%20relacion%20entre%20html%2C%20css%20y%20javascript_images/image-002.png)

**⚙️ 3. JavaScript: la interacción**

JavaScript es el lenguaje que añade **comportamiento dinámico** a la página.

**Qué aporta:**

-   Responder a eventos (clics, teclas, scroll).
-   Modificar el HTML y CSS en tiempo real.
-   Validar formularios.
-   Crear aplicaciones completas (SPA).
-   Conectar con servidores (APIs).

**Ejemplo:**

![](./02-%20relacion%20entre%20html%2C%20css%20y%20javascript_images/image-003.png)

**🔗 ¿Cómo se relacionan entre sí?**

**🧩 1. HTML es la base**

CSS y JavaScript **necesitan** HTML para actuar sobre él. Sin elementos HTML, no hay nada que estilizar ni manipular.

**🎨 2. CSS se aplica sobre HTML**

CSS selecciona elementos HTML y les da estilo. Ejemplo:

![](./02-%20relacion%20entre%20html%2C%20css%20y%20javascript_images/image-004.png)

→ afecta a todos los <p> del HTML.

**⚙️ 3. JavaScript manipula HTML y CSS**

JavaScript puede:

-   Crear o eliminar elementos HTML.
-   Cambiar estilos CSS.
-   Añadir clases.
-   Modificar atributos.

Ejemplo:

![](./02-%20relacion%20entre%20html%2C%20css%20y%20javascript_images/image-005.png)

**🔄 4. Trabajan juntos para crear experiencias completas**

-   HTML → contenido
-   CSS → diseño
-   JavaScript → interacción

Sin uno de ellos, la web sería incompleta.

**🖼️ Metáfora para entenderlo mejor**

Imagina que estás construyendo una casa:

| **Tecnología** | **Equivalencia** | **Función** |
| --- | --- | --- |
| **HTML** | Estructura de la casa | Paredes, puertas, habitaciones |
| **CSS** | Decoración | Colores, muebles, estilos |
| **JavaScript** | Electricidad y mecanismos | Luces, interruptores, puertas automáticas |

**🎯 Conclusión**

HTML, CSS y JavaScript forman un **equipo inseparable**:

-   **HTML** estructura
-   **CSS** embellece
-   **JavaScript** da vida

Juntos permiten crear desde páginas simples hasta aplicaciones web complejas.
