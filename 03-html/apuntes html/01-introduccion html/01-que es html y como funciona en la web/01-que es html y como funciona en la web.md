**1\. QUE ES HTML Y CÓMO FUNCIONA EN LA WEB**

**🟦 1. ¿Qué es HTML?**

**HTML (HyperText Markup Language)** es el **lenguaje estándar** que se utiliza para crear páginas web. No es un lenguaje de programación, sino un **lenguaje de marcado**, lo que significa que usa **etiquetas** para estructurar y describir el contenido.

**🔹 ¿Para qué sirve?**

-   Define **qué contenido** aparece en una página (texto, imágenes, enlaces, tablas, formularios…).
-   Organiza ese contenido en una **estructura lógica**.
-   Permite que los navegadores (Chrome, Firefox, Edge…) **interpreten y muestren** la página correctamente.

**🟩 2. Cómo funciona HTML en la web**

**🌐 2.1. El navegador interpreta el HTML**

Cuando visitas una página web:

1.  Escribes una URL o haces clic en un enlace.
2.  El navegador solicita el archivo HTML al servidor.
3.  El servidor envía el archivo.
4.  El navegador **lee el HTML** y construye la página que ves en pantalla.

Este proceso se llama **renderizado**.

**🧱 2.2. HTML define la estructura de la página**

HTML organiza el contenido en bloques mediante **etiquetas** como:

| **Etiqueta HTML** | **Función** |
| --- | --- |
| &lt;h1&gt; | Título principal |
| &lt;p&gt; | Párrafo |
| &lt;img&gt; | Imagen |
| &lt;a&gt; | Enlace |
| &lt;div&gt; | Contenedor general |
| &lt;header&gt; | Cabecera |
| &lt;footer&gt; | Pie de página |

**🎨 2.3. HTML trabaja junto a CSS y JavaScript**

HTML **no trabaja solo**. Forma parte del trío fundamental de la web:

| **Tecnología** | **Función** |
| --- | --- |
| **HTML** | Estructura el contenido |
| **CSS** | Da estilo (colores, tamaños, diseño) |
| **JavaScript** | Añade interactividad y lógica |

Ejemplo de cómo se complementan:

-   HTML crea un botón.
-   CSS lo hace bonito.
-   JavaScript hace que funcione al hacer clic.

**🟧 3. Características importantes de HTML**

**✔ Es un lenguaje declarativo**

Describe **qué es** cada elemento, no cómo debe comportarse.

**✔ Es extensible**

Puedes añadir atributos para dar más información:

![](./01-que%20es%20html%20y%20como%20funciona%20en%20la%20web_images/image-002.png)

**✔ Es la base de toda página web**

Sin HTML, no habría estructura ni contenido.

**🟨 4. ¿Por qué es tan importante?**

-   Es el **primer paso** para aprender desarrollo web.
-   Todos los navegadores lo entienden.
-   Permite crear páginas accesibles y bien organizadas.
-   Es fácil de aprender pero muy poderoso cuando se combina con CSS y JS.

**📝 5. Resumen final**

-   HTML es el **lenguaje que estructura** las páginas web.
-   Funciona mediante **etiquetas** que indican qué representa cada parte del contenido.
-   El navegador **lee el HTML** y construye la página.
-   Trabaja junto a **CSS** (estilos) y **JavaScript** (interactividad).
-   Es esencial para cualquier proyecto web.
