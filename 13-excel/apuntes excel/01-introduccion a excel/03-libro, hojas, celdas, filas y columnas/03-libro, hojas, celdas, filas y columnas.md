**📘 3. Libro, hojas, celdas, filas y columnas**

Para trabajar con Excel es fundamental entender cómo se estructura un archivo. Excel organiza la información de forma jerárquica, desde el archivo completo (libro) hasta la unidad mínima (celda). Dominar estos conceptos te permitirá moverte con soltura y comprender cómo funcionan las fórmulas y los datos.

**🟦 3.1. Libro de Excel**

El **libro** es el archivo completo de Excel. Su extensión suele ser **.xlsx** (o .xls en versiones antiguas).

**✔️ Características del libro:**

-   Contiene una o varias **hojas de cálculo**.
-   Puedes añadir, eliminar, renombrar o mover hojas.
-   Toda la información (tablas, gráficos, fórmulas…) se guarda dentro del libro.
-   Puedes tener varios libros abiertos al mismo tiempo.

**✔️ ¿Para qué sirve el libro?**

-   Para organizar diferentes conjuntos de datos dentro de un mismo archivo.
-   Para separar información por categorías, meses, departamentos, etc.
-   Para mantener todo el proyecto en un único documento.

**🟦 3.2. Hojas de cálculo**

Una **hoja** es cada una de las páginas dentro del libro. Por defecto, Excel abre un libro con una hoja llamada “Hoja1”.

**✔️ Elementos de una hoja:**

-   Una cuadrícula formada por **filas** y **columnas**.
-   Celdas donde se introducen datos.
-   Pestañas en la parte inferior para cambiar entre hojas.

**✔️ Acciones comunes con las hojas:**

-   Renombrar (doble clic en la pestaña).
-   Cambiar color de la pestaña.
-   Mover o copiar hojas.
-   Insertar nuevas hojas.
-   Eliminar hojas que no se necesiten.

**✔️ ¿Para qué sirven las hojas?**

-   Para separar información dentro del mismo archivo.
-   Para organizar datos por temas (por ejemplo: ventas, gastos, inventario).
-   Para mantener cálculos o informes independientes pero relacionados.

**🟦 3.3. Columnas**

Las **columnas** son las divisiones verticales de la hoja. Se identifican con **letras**: A, B, C… Z, AA, AB, etc.

**✔️ Características:**

-   Cada columna contiene miles de celdas.
-   Puedes ajustar su ancho manualmente o automáticamente.
-   Puedes ocultar columnas si no quieres que se vean.

**✔️ ¿Para qué sirven?**

-   Para organizar datos por categorías. Ejemplo: *Nombre*, *Edad*, *Ciudad*, *Precio*, *Cantidad*.

**🟦 3.4. Filas**

Las **filas** son las divisiones horizontales de la hoja. Se identifican con **números**: 1, 2, 3, 4…

**✔️ Características:**

-   Cada fila contiene celdas alineadas horizontalmente.
-   Puedes ajustar su altura.
-   Puedes ocultar filas si no quieres mostrarlas.

**✔️ ¿Para qué sirven?**

-   Para organizar registros o entradas de datos. Ejemplo: cada fila puede representar un cliente, un producto o un día.

**🟦 3.5. Celdas**

La **celda** es la unidad mínima de Excel. Es el espacio donde introduces datos, textos, números o fórmulas.

**✔️ Identificación de una celda**

Cada celda se identifica por su **columna + fila**. Ejemplos:

-   A1 → columna A, fila 1
-   C5 → columna C, fila 5
-   F12 → columna F, fila 12

A esto se le llama **referencia de celda**.

**✔️ ¿Qué puede contener una celda?**

-   Texto (nombres, descripciones)
-   Números
-   Fechas y horas
-   Fórmulas
-   Resultados de cálculos
-   Comentarios o notas

**✔️ Propiedades importantes:**

-   Puedes aplicar formato (color, bordes, alineación).
-   Puedes combinar varias celdas en una sola.
-   Puedes bloquear celdas para proteger datos.
-   Las fórmulas siempre se escriben empezando por **\=**.

**🟦 Resumen rápido para estudiar**

-   **Libro** → el archivo completo.
-   **Hojas** → páginas dentro del libro.
-   **Columnas** → verticales, identificadas por letras.
-   **Filas** → horizontales, identificadas por números.
-   **Celdas** → intersección de fila y columna; donde se introducen datos
