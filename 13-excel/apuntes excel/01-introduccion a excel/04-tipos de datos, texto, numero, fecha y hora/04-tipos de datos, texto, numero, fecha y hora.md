**📘 4. Tipos de datos: texto, número, fecha y hora**

Excel puede trabajar con distintos tipos de datos. Reconocerlos es esencial para evitar errores y para que las fórmulas funcionen correctamente. Cada tipo de dato tiene un comportamiento propio dentro de las celdas.

**🟦 4.1. Texto**

El **texto** es cualquier dato que Excel no interpreta como número, fecha u hora. Incluye palabras, códigos, combinaciones de letras y números, o números escritos como texto.

**✔️ Características del texto:**

-   Se alinea **a la izquierda** por defecto.
-   No se puede usar directamente en operaciones matemáticas.
-   Excel lo trata como información descriptiva.
-   Puede contener letras, números y símbolos.

**✔️ Ejemplos:**

-   “Nombre”
-   “Producto A”
-   “123A”
-   “Málaga”
-   “00123” (si está como texto, Excel no lo convierte en número)

**✔️ Cuándo usar texto:**

-   Para nombres, direcciones, códigos, descripciones.
-   Para números que no deben calcularse (DNI, códigos postales, matrículas).

**🟦 4.2. Número**

Los **números** son valores que Excel puede usar en cálculos. Incluyen enteros, decimales, porcentajes y valores negativos.

**✔️ Características de los números:**

-   Se alinean **a la derecha** por defecto.
-   Se pueden usar en fórmulas y funciones.
-   Pueden tener formato especial (moneda, porcentaje, contabilidad, etc.).
-   Excel reconoce automáticamente la mayoría de números.

**✔️ Ejemplos:**

-   25
-   3,14
-   \-8
-   1500
-   0,25 (que puede mostrarse como 25%)

**✔️ Cuándo usar números:**

-   Para cantidades, precios, totales, cálculos, estadísticas.

**🟦 4.3. Fecha**

Las **fechas** en Excel son números en realidad. Excel las interpreta como un valor numérico que representa días desde una fecha base (normalmente 1/1/1900).

**✔️ Características de las fechas:**

-   Se alinean **a la derecha** por defecto.
-   Se pueden usar en cálculos (sumar días, restar fechas, etc.).
-   Se pueden mostrar en distintos formatos:
    -   31/03/2026
    -   31-03-26
    -   Martes, 31 de marzo de 2026

**✔️ Ejemplos:**

-   01/01/2024
-   15-08-2025
-   31/03/2026

**✔️ Importante:**

Si escribes una fecha en un formato que Excel no reconoce, la tratará como **texto**.

**✔️ Cuándo usar fechas:**

-   Para calendarios, registros, vencimientos, planificación, informes.

**🟦 4.4. Hora**

Las **horas** también son números para Excel. Representan una fracción del día (por ejemplo, 12:00 es 0,5).

**✔️ Características de las horas:**

-   Se alinean **a la derecha** por defecto.
-   Se pueden sumar, restar y comparar.
-   Pueden mostrarse en distintos formatos:
    -   14:30
    -   14:30:00
    -   2:30 PM (según configuración)

**✔️ Ejemplos:**

-   08:00
-   14:30
-   23:59

**✔️ Cuándo usar horas:**

-   Para horarios, turnos, duración de actividades, control de tiempo.

**🟦 Resumen rápido para estudiar**

-   **Texto** → información descriptiva, no se calcula.
-   **Número** → valores para operaciones matemáticas.
-   **Fecha** → números que representan días; permiten cálculos de tiempo.
-   **Hora** → fracciones del día; permiten cálculos de duración.
