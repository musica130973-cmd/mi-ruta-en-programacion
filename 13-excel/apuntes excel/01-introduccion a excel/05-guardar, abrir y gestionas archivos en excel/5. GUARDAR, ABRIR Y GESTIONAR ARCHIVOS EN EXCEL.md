**📘 5. Guardar, abrir y gestionar archivos en Excel**

Trabajar con Excel no solo implica introducir datos o usar fórmulas; también es fundamental saber **guardar, abrir y organizar tus archivos** correctamente. Esto garantiza que no pierdas información y que puedas trabajar de forma ordenada y profesional.

**🟦 5.1. Guardar un archivo**

Guardar un archivo es una de las acciones más importantes. Excel ofrece varias formas de hacerlo.

**✔️ Guardar por primera vez**

Cuando guardas un archivo nuevo:

1.  Haz clic en **Archivo** (esquina superior izquierda).
2.  Selecciona **Guardar como**.
3.  Elige la ubicación (PC, OneDrive, carpeta específica).
4.  Escribe un nombre para el archivo.
5.  Selecciona el tipo de archivo (por defecto: **.xlsx**).
6.  Haz clic en **Guardar**.

**✔️ Atajo de teclado**

-   **Ctrl + G** → Guardar
-   **Ctrl + S** → Guardar (más usado)

**✔️ Guardar cambios en un archivo existente**

Si el archivo ya tiene nombre y ubicación:

-   Basta con pulsar **Ctrl + S** o hacer clic en el icono de guardar.

**✔️ Guardar con otro nombre**

Útil para crear versiones diferentes del mismo archivo.

-   Archivo → **Guardar como** → Cambiar nombre o ubicación.

**🟦 5.2. Tipos de archivo más comunes en Excel**

Excel permite guardar en distintos formatos según tus necesidades:

| **Formato** | **Extensión** | **Uso principal** |
| --- | --- | --- |
| **Libro de Excel** | .xlsx | Formato estándar, sin macros |
| **Libro habilitado para macros** | .xlsm | Permite guardar macros |
| **Libro de Excel 97-2003** | .xls | Compatibilidad con versiones antiguas |
| **CSV (valores separados por comas)** | .csv | Exportar datos sin formato |
| **PDF** | .pdf | Compartir informes sin editar |

**🟦 5.3. Abrir un archivo**

Para abrir un archivo de Excel:

**✔️ Método 1: Desde Excel**

1.  Archivo → **Abrir**
2.  Selecciona la ubicación
3.  Elige el archivo
4.  Haz clic en **Abrir**

**✔️ Método 2: Desde el explorador de archivos**

-   Doble clic sobre el archivo .xlsx
-   Excel se abrirá automáticamente

**✔️ Método 3: Arrastrar y soltar**

-   Arrastra el archivo desde una carpeta y suéltalo dentro de Excel

**🟦 5.4. Gestionar archivos dentro de Excel**

Excel ofrece varias herramientas para organizar y trabajar con tus documentos.

**✔️ Crear un archivo nuevo**

-   Archivo → **Nuevo**
-   Elegir “Libro en blanco” o una plantilla

**✔️ Cerrar un archivo**

-   Archivo → **Cerrar**
-   Si hay cambios sin guardar, Excel te avisará

**✔️ Recuperar archivos no guardados**

Excel guarda versiones temporales automáticamente.

Ruta:

-   Archivo → **Información** → **Administrar libro** → *Recuperar libros no guardados*

Muy útil si se cierra Excel por error.

**🟦 5.5. Ubicaciones de guardado**

Puedes guardar tus archivos en:

**✔️ En tu ordenador**

-   Carpetas locales (Documentos, Escritorio, etc.)

**✔️ En la nube (OneDrive)**

Ventajas:

-   Copia de seguridad automática
-   Acceso desde cualquier dispositivo
-   Facilita compartir archivos

**✔️ Unidades externas**

-   Pendrive
-   Disco duro externo

**🟦 5.6. Buenas prácticas para gestionar archivos**

-   Usa nombres claros y descriptivos Ejemplo: *Ventas\_2026\_Marzo.xlsx*
-   Organiza tus archivos en carpetas
-   Guarda versiones importantes Ejemplo: *Proyecto\_v1*, *Proyecto\_v2*
-   Haz copias de seguridad periódicas
-   Evita usar caracteres no permitidos (/, \*, ?, etc.)

**🟦 Resumen rápido para estudiar**

-   **Guardar**: Ctrl + S, elegir nombre y ubicación.
-   **Abrir**: desde Excel, doble clic o arrastrar.
-   **Gestionar**: crear, cerrar, recuperar versiones.
-   **Formatos**: .xlsx (normal), .xlsm (macros), .csv (datos simples).
-   **Buenas prácticas**: nombres claros, carpetas organizadas, copias de seguridad.
