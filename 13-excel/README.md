📊 Excel
Bienvenido a mi sección dedicada a Microsoft Excel, una herramienta esencial para el análisis de datos, la gestión de información y la creación de informes profesionales.
Aquí recopilo mis apuntes, prácticas, funciones clave, ejercicios y proyectos, siguiendo un temario progresivo que va desde cero hasta un nivel avanzado.

Este espacio está pensado tanto para mi propio aprendizaje como para cualquiera que quiera dominar Excel de forma práctica y estructurada.

📘 ¿Qué encontrarás en esta carpeta?
Los contenidos están organizados por módulos, cada uno con su propio archivo en formato:

.md → lectura rápida desde GitHub

.xlsx / .docx → ejercicios prácticos y versiones más visuales para estudiar o imprimir

Incluyen teoría, ejemplos, funciones, actividades y ejercicios paso a paso.
