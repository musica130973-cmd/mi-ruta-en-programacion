**✅ SOLUCIONES – Tema 2**

**✔️ Solución Actividad 2 (relacionar conceptos)**

1 → E (Cinta de opciones → barra superior con herramientas) 2 → C (Pestañas → categorías) 3 → F (Grupos → subdivisiones dentro de pestañas) 4 → A (Comandos → botones) 5 → B (Barra de fórmulas → área para escribir/editar) 6 → D (Cuadro de nombre → referencia de celda)

**✔️ Solución Actividad 3 (clasificar herramientas)**

1.  Autosuma → **Inicio**
2.  Insertar gráfico → **Insertar**
3.  Validación de datos → **Datos**
4.  Ortografía → **Revisar**
5.  Inmovilizar paneles → **Vista**
6.  Insertar tabla → **Insertar**
7.  Quitar duplicados → **Datos**
8.  Comentarios → **Revisar**

**✔️ Solución Actividad 5 (preguntas de comprensión)**

1.  Porque así es más fácil encontrar herramientas agrupadas por función.
2.  Permite tener más espacio en pantalla para ver la hoja.
3.  En la celda ves el **resultado**; en la barra de fórmulas ves el **contenido real** (texto o fórmula).
4.  Sirve para saber exactamente qué celda está activa y para nombrar rangos.
