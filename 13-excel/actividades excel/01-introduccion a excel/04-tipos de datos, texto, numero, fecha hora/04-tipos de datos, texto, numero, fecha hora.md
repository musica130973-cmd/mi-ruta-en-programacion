**✔️ Solución Actividad 4**

Excel no puede usarlos porque están **entre comillas**, por lo que los interpreta como **texto**.

-   “25” → debería ser → 25
-   “00150” → debería ser → 150
-   “3,14” → debería ser → 3,14

**✔️ Solución Actividad 5**

1.  31/02/2024 → **Texto** (fecha inexistente)
2.  15/08/2025 → **Fecha válida**
3.  2026/03/31 → **Texto** (formato no reconocido en muchas configuraciones)
4.  31-03-2026 → **Fecha válida**
5.  “31/03/2026” → **Texto** (comillas = texto)

**✔️ Solución Actividad 6**

3.  Porque representan una fracción del día (ej.: 12:00 = 0,5).
4.  Permite hacer cálculos con números y mantener el texto como información descriptiva.
