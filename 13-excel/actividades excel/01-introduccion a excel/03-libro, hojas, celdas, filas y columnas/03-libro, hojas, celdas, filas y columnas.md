**✅ SOLUCIONES – Tema 3**

**✔️ Solución Actividad 1 (descripciones orientativas)**

-   **Libro:** archivo completo de Excel que contiene varias hojas.
-   **Hojas:** páginas dentro del libro donde se introducen datos.
-   **Pestañas de hojas:** botones en la parte inferior para cambiar entre hojas.
-   **Filas:** divisiones horizontales numeradas.
-   **Columnas:** divisiones verticales identificadas por letras.
-   **Celdas:** intersección entre una fila y una columna; unidad mínima donde se escribe.

**✔️ Solución Actividad 2 (relacionar conceptos)**

-   Libro → **B**
-   Hoja → **D**
-   Columna → **C**
-   Fila → **E**
-   Celda → **A**

**✔️ Solución Actividad 3 (referencias de celdas)**

-   C7
-   A15
-   F3
-   J20
-   B1

**✔️ Solución Actividad 4 (pasos orientativos)**

1.  Insertar hoja → botón “+” o clic derecho → “Insertar”.
2.  Renombrar → doble clic en la pestaña → escribir nombre.
3.  Cambiar color → clic derecho → “Color de pestaña”.
4.  Mover hoja → arrastrar la pestaña a la izquierda.
5.  Duplicar hoja → clic derecho → “Mover o copiar” → “Crear una copia”.

**✔️ Solución Actividad 5 (clasificación)**

-   “Cliente 1, Cliente 2, Cliente 3…” → **Fila**
-   “Nombre, Apellidos, Dirección…” → **Columna**
-   “A5” → **Celda**
-   “C12” → **Celda**
-   “1, 2, 3, 4…” → **Fila**
-   “A, B, C, D…” → **Columna**

**✔️ Solución Actividad 6 (preguntas de comprensión)**

1.  Para separar información sin crear varios archivos.
2.  Permite localizar cualquier celda de forma exacta.
3.  La fila es horizontal y se numera; la columna es vertical y se identifica con letras.
4.  Porque Excel necesita “=” para entender que debe calcular algo.

**✔️ Solución Actividad 7 (mini‑práctica)**

Al combinar celdas:

-   Varias celdas se unen en una sola.
-   Solo queda activa la celda superior izquierda.
-   El contenido se centra en el área combinada.
-   Las celdas individuales dejan de existir visualmente.
