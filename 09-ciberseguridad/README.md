
🛡️ Ciberseguridad
Bienvenido a mi sección dedicada a la ciberseguridad, un área esencial en el mundo tecnológico actual.
Aquí recopilo mis apuntes, prácticas, conceptos clave, herramientas y metodologías para formarme desde los fundamentos hasta un nivel profesional.

Este espacio está pensado tanto para mi propio aprendizaje como para cualquiera que quiera iniciarse o profundizar en la seguridad informática.

📘 ¿Qué encontrarás en esta carpeta?
Los contenidos están organizados por temas, cada uno con su propio archivo en formato:

.md → lectura rápida desde GitHub

.docx → versión más visual para estudiar o imprimir

Incluyen teoría, ejemplos, comandos, esquemas y ejercicios prácticos.
