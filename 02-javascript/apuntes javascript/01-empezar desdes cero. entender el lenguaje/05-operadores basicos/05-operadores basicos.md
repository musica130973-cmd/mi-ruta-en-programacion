**📝 Resumen para estudiar**

-   **Los operadores aritméticos realizan cálculos.**
-   **Los de asignación modifican el valor de una variable.**
-   **Los de comparación devuelven booleanos y se usan en condicionales.**
-   **Los lógicos combinan condiciones.**
-   **typeof e instanceof ayudan a identificar tipos.**
-   **El operador ternario simplifica decisiones.**
