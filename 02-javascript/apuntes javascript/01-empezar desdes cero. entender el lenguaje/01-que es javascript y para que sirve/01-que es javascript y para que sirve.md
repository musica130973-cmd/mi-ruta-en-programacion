**1\. QUÉ ES JAVASCRIPT Y PARA QUÉ SIRVE**

**🌐 Qué es JavaScript**

JavaScript es un lenguaje **ligero**, **dinámico** y **multiparadigma** (soporta programación imperativa, funcional y orientada a objetos). Se ejecuta normalmente en el navegador, pero también en entornos externos como **Node.js**, lo que permite usarlo tanto en el cliente como en el servidor.

Sus características clave incluyen:

-   **Interpretado o JIT**: el navegador o motor lo ejecuta directamente sin necesidad de compilación previa.
-   **Dinámico**: los tipos se determinan en tiempo de ejecución.
-   **Basado en prototipos**: los objetos heredan directamente de otros objetos.
-   **Recolección de basura**: gestiona la memoria automáticamente.
-   **Eventos y asincronía**: ideal para interfaces reactivas y aplicaciones en tiempo real.

**🧩 Para qué sirve JavaScript**

JavaScript nació para hacer las páginas web interactivas, pero su uso se ha expandido enormemente. Hoy sirve para:

**Desarrollo web en el navegador**

Es el lenguaje que permite que una web **reaccione** a las acciones del usuario: animaciones, validación de formularios, menús dinámicos, efectos visuales, actualizaciones sin recargar la página, etc. Es el “lenguaje de la web”.

**Desarrollo del lado del servidor**

Con **Node.js**, JavaScript puede crear servidores completos, APIs, aplicaciones en tiempo real (chats, juegos online) y sistemas escalables.

**Aplicaciones móviles**

Frameworks como **React Native**, **Ionic** o **NativeScript** permiten crear apps para Android e iOS usando JavaScript.

**Aplicaciones de escritorio**

Herramientas como **Electron** permiten crear programas de escritorio (como VS Code) con tecnologías web.

**Videojuegos**

Motores como **Phaser** o **Babylon.js** permiten crear juegos 2D y 3D directamente en el navegador.

**Automatización y herramientas**

JavaScript se usa para scripts, automatización de tareas, herramientas de desarrollo y utilidades CLI.

**🧠 Cómo funciona JavaScript en el navegador**

Cuando una página web se carga:

1.  El navegador interpreta el HTML y el CSS.
2.  Cuando encuentra JavaScript, lo ejecuta mediante un motor (como **V8**, **SpiderMonkey** o **JavaScriptCore**).
3.  El código puede modificar el contenido de la página (DOM), reaccionar a eventos y comunicarse con servidores.

Este modelo permite crear experiencias web dinámicas y modernas.

**📌 Por qué JavaScript es tan importante hoy**

-   Es el **único lenguaje nativo del navegador**.
-   Tiene un ecosistema enorme: React, Angular, Vue, Node.js, Express, etc.
-   Permite usar el mismo lenguaje en **frontend y backend**.
-   Es flexible, rápido de aprender y muy demandado en el mercado laboral.

**📝 Resumen para estudiar**

-   JavaScript es un lenguaje **interpretado, dinámico y multiparadigma**.
-   Se usa para **interactividad web**, pero también para servidores, apps móviles, escritorio y juegos.
-   Funciona en navegadores y en entornos como **Node.js**.
-   Es uno de los lenguajes más influyentes y utilizados del mundo.
